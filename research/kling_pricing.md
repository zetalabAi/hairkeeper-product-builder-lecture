## 크레딧 소모량

### 이미지 생성
- **일반 이미지**: ~0.2 크레딧/이미지
- **고품질 이미지**: ~1 크레딧/이미지

### 영상 생성
- **Kling 2.6 Standard 5s**: ~20 크레딧/영상
- **Kling 2.6 Pro 5s**: ~35 크레딧/영상
- **Kling O1 (Element AI)**: 더 높은 크레딧 소모
- **1080p 영상 확장**: 추가 크레딧 필요

### Face Swap (추정)
- **이미지 Face Swap**: ~1-5 크레딧/이미지
- **영상 Face Swap**: 영상 생성 비용과 동일 (~20-35 크레딧/5초)

## 주요 기능

### Standard 플랜
- 660 크레딧/월
- 브랜드 워터마크 제거
- 상업적 사용 가능
- 빠른 생성 (일 3회)
- VIDEO O1 모델 (Element AI, Multi-Shot)
- VIDEO 2.6 모델 (Voice Control)

### Pro 플랜
- 3,000 크레딧/월
- 모든 Standard 기능
- 빠른 생성 무제한
- 1080p 영상 생성
- 이미지 업스케일링
- 우선 접근

### Premier 플랜
- 8,000 크레딧/월
- 모든 Pro 기능
- 영상 확장 기능
- 베타 기능 테스트 초대

### Ultimate 플랜
- 26,000 크레딧/월
- 모든 Premier 기능
- 무제한 대기열
- 최우선 지원

- 명시되지 않음 (추가 확인 필요)

- 공식 웹사이트에서 크레딧 카드 구매 가능

## 상업적 사용
- **Free 플랜**: 제한적
- **유료 플랜**: 모두 상업적 사용 가능

## 워터마크
- **Free 플랜**: 워터마크 포함
- **유료 플랜**: 브랜드 워터마크 제거

- **Daily 3 Free Uses**: 모든 유료 플랜에서 빠른 생성 일 3회 무료 (또는 무제한)
- **Referral Bonus**: 첫 달 50% 보너스 크레딧 (프로모션)
