## Face Swap 예상 비용

### 이미지 합성 (Head Swap)
- **예상 크레딧**: ~10-20 크레딧/이미지
- **Creator 플랜 기준**: 월 150-300장 생성 가능
- **비용**: $24.99/월

### 영상 생성 (5초 릴스)
- **Dzine V2 720p 5s**: ~20 크레딧/영상
- **Kling 2.6 1080p 5s**: ~85 크레딧/영상
- **Creator 플랜 기준**: 월 35-150개 생성 가능

## 크레딧 롤오버
- **Creator/Master 플랜**: 이미지 크레딧은 최대 3개월까지 이월 가능
- **영상 크레딧**: 이월 불가

## 추가 크레딧 구매
- 공식 웹사이트에서 명시되지 않음
- 플랜 업그레이드로 크레딧 확보 권장

## 상업적 사용
- **Free 플랜**: 비상업적 사용만 가능
- **Beginner/Creator/Master**: 상업적 사용 가능

## 워터마크
- **Free 플랜**: 워터마크 포함
- **유료 플랜**: 워터마크 제거

## API 가격
- 공식 API 페이지 존재 (<https://www.dzine.ai/api/>)
- 구체적인 가격 정보는 별도 문의 필요
