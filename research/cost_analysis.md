# 머리보존 AI - 비용 구조 분석 및 마진 계산

## 1. 사용자 요구사항 정리

### 서비스 구조
- **9명의 얼굴 세트** (각 얼굴당 상하좌우 다각도 포함)
- **이미지 합성**: 사용자 사진 + 선택한 얼굴 → 합성 이미지
- **영상 생성**: 합성 이미지 → 5초 릴스 영상

### 무료 서비스
- **최초 3회**: 이미지 합성 + 영상 생성 (워터마크 포함)
- **매주 월요일 오전 9시**: 2회 충전
  - 이미지 합성만 가능
  - 영상 생성 불가
  - 워터마크 포함

### 유료 서비스
- **프로**: 이미지 합성만, 워터마크 제거
- **프로플러스**: 이미지 합성 + 영상 생성, 워터마크 제거
- **크레딧 소모제**: 월 구독 + 크레딧 소모
- **추가 크레딧 구매**: 크레딧 소진 시 추가 구매 가능
- **다운그레이드**: 언제나 가능

## 2. Dzine AI vs Kling AI 비교

| 항목 | Dzine AI | Kling AI |
|------|----------|----------|
| **API 비용** | ~$0.10-0.20/이미지 | ~$0.01-0.05/이미지 |
| **품질** | 매우 우수 (머리카락 보존 완벽) | 우수 |
| **처리 속도** | 중간 (~30초) | 빠름 (~10초) |
