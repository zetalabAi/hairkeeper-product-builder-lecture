# Beta Testing Checklist

베타 테스트를 성공적으로 진행하기 위한 체크리스트입니다.

## 배포 전 확인

### Firebase 설정
- [ ] `.env.production.example`을 `.env.production`으로 복사
- [ ] 실제 API 키로 플레이스홀더 교체
  - [ ] `ONEMIN_AI_API_KEY`
  - [ ] `DZINE_AI_API_KEY`
  - [ ] `FIREBASE_PROJECT_ID`
  - [ ] `GOOGLE_APPLICATION_CREDENTIALS`
- [ ] Firebase Functions 배포
  ```bash
  firebase deploy --only functions
  ```
- [ ] Firestore 보안 규칙 배포
  ```bash
  firebase deploy --only firestore:rules
  ```
- [ ] Firebase Console에서 배포 확인
  - [ ] Functions 정상 작동
  - [ ] Firestore 규칙 적용됨

### 얼굴 풀 준비
- [ ] 한국인 여성 얼굴 7장 업로드 완료
- [ ] 한국인 남성 얼굴 7장 업로드 완료
- [ ] Firestore `facePool` 컬렉션 확인
- [ ] 모든 이미지 URL 접근 가능
- [ ] 메타데이터 정확성 확인 (gender, style)

### 앱 빌드
- [ ] EAS 계정 설정
- [ ] iOS 베타 빌드
  ```bash
  eas build --platform ios --profile preview
  ```
- [ ] Android 베타 빌드
  ```bash
  eas build --platform android --profile preview
  ```
- [ ] TestFlight에 iOS 빌드 업로드
- [ ] Google Play Internal Testing에 Android 빌드 업로드

## 기능 테스트

### 핵심 기능
- [ ] **사진 선택**
  - [ ] 카메라로 촬영
  - [ ] 갤러리에서 선택
  - [ ] 이미지 크롭 가능
- [ ] **성별 선택**
  - [ ] 남성/여성 선택 UI 표시
  - [ ] 선택 후 다음 단계 진행
- [ ] **얼굴 선택**
  - [ ] 얼굴 풀 로딩 (진행률 표시)
  - [ ] 이미지 프리로딩 완료
  - [ ] 얼굴 클릭 시 즉시 표시
  - [ ] 선택된 얼굴 하이라이트
- [ ] **얼굴 합성**
  - [ ] 진행률 표시 (0-100%)
  - [ ] 처리 메시지 표시
  - [ ] 20-25초 이내 완료 (balanced 모드)
  - [ ] 에러 발생 시 적절한 메시지
- [ ] **결과 확인**
  - [ ] 원본/결과 이미지 표시
  - [ ] 이미지 확대/축소 가능
  - [ ] 공유 버튼 작동
  - [ ] 다운로드 버튼 작동

### 배치 처리
- [ ] 여러 이미지 선택 (최대 10장)
- [ ] 병렬 처리 진행률 표시
- [ ] 모든 이미지 처리 완료
- [ ] ZIP 다운로드 버튼 표시
- [ ] ZIP 파일 다운로드 및 압축 해제
- [ ] 모든 결과 이미지 포함 확인

### 베타 피드백
- [ ] 피드백 화면 접근
- [ ] 6개 정량 평가 입력
- [ ] 3개 정성 평가 입력
- [ ] 제출 버튼 활성화 (모든 필드 입력 후)
- [ ] 제출 완료 메시지
- [ ] Firestore `betaFeedback` 컬렉션 저장 확인

## 성능 테스트

### 이미지 최적화
- [ ] 대용량 이미지 (5MB+) 업로드
- [ ] 자동 리사이징 확인 (1080px)
- [ ] 파일 크기 감소 확인 (60-80%)
- [ ] 업로드 시간 2초 이내

### 프리로딩
- [ ] 얼굴 풀 진입 시 로딩 표시
- [ ] 진행률 0% → 100% 표시
- [ ] 로딩 완료 후 즉시 이미지 표시
- [ ] 스크롤 시 깜빡임 없음

### API 처리 시간
- [ ] Fast 모드: 15-20초
- [ ] Balanced 모드: 20-25초
- [ ] High 모드: 30-35초
- [ ] 타임아웃 처리 (2분)

### 배치 처리 성능
- [ ] 10장 이미지 동시 처리
- [ ] 병렬 처리 (최대 4개 동시)
- [ ] 전체 처리 시간 5분 이내
- [ ] 메모리 누수 없음
