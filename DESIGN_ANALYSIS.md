# 기존 디자인 분석 (영상 기반)

## 전체 테마
- **다크 모드**: 검은색 배경 (#000000 또는 매우 어두운 회색)
- **Primary Color**: 보라색/퍼플 (#8B7FF5 계열)
- **텍스트**: 흰색 (제목), 회색 (설명)

## 화면별 디자인 특징

### 1. 홈 화면
- **배경**: 순수 검은색
- **헤더**: 
  - 왼쪽: "머리보존" 텍스트
  - 오른쪽: 홈 아이콘
- **메인 액션 버튼**:
  - 보라색 둥근 사각형 버튼 (rounded-2xl)
  - 중앙에 + 아이콘
  - 하단에 "새 프로젝트 시작" 텍스트
  - 설명 텍스트: "사진을 선택하여 얼굴 교체 작업을 시작하세요"
- **최근 작업물 섹션**:
  - "최근 작업물" 제목
  - 빈 상태: "아직 작업물이 없습니다\n새 프로젝트를 시작해보세요"
- **하단 탭바**: 
  - 보라색 홈 아이콘 (활성)

### 2. 사진 선택 화면
- **헤더**: 
  - 왼쪽: 뒤로가기 아이콘 (<>)
  - 중앙: "사진 선택" 텍스트
- **탭 버튼**:
  - "사진" / "컬렉션" 토글
  - 선택된 탭: 보라색 배경
  - 비선택 탭: 어두운 회색
- **권한 요청 UI**:
  - 아이콘 + 설명 텍스트
  - "Expo Go에서 내가 선택한 사진에만 액세스합니다"
  - 오른쪽 하단: "닫기" 버튼

### 3. 스타일 선택 화면
- **헤더**: 
  - 왼쪽: 뒤로가기 (<>)
  - 중앙: "스타일 선택"
- **설명**: "원하는 얼굴 스타일을 선택하세요"
- **옵션 그리드**:
  - 2열 레이아웃
  - 각 옵션: 어두운 배경 카드
  - 아이콘 (집 모양 - 실제로는 다른 아이콘 사용 예정)
  - 하단에 텍스트 레이블
  - 옵션: 청순, 귀여움, 아름다움, 도도, 섹시
  - 선택된 항목: 보라색 배경 + 체크 아이콘
- **하단 버튼**:
  - 전체 너비 보라색 버튼
  - "얼굴 추천받기" 텍스트
- **하단 설명**: "선택한 스타일에 맞는 얼굴 풀에서 추천됩니다"

### 4. 로딩 화면
- **중앙 정렬**:
  - 보라색 로딩 스피너
  - "얼굴을 합성하는 중..."
  - "머리는 절대 변형되지 않습니다" (작은 텍스트)

### 5. 결과 확인 화면
- **헤더**:
  - 왼쪽: 뒤로가기 (<>)
  - 중앙: "결과 확인"
- **이미지 비교 UI**:
  - 좌우 분할 슬라이더
  - 왼쪽: "원본 이미지"
  - 오른쪽: "성 결과" (합성 결과)
  - 중앙에 보라색 세로 바 + 슬라이더 핸들
- **설명**: "좌우로 드래그하여 원본과 결과를 비교하세요"
- **체크 메시지**: "✓ 머리는 절대 변형되지 않았습니다"
- **하단 버튼**:
  - 전체 너비 보라색 버튼
  - "저장하기" 텍스트

## 주요 디자인 원칙

1. **다크 테마 우선**: 모든 화면이 검은색 배경
2. **보라색 액센트**: Primary 색상으로 일관되게 사용
3. **큰 터치 영역**: 버튼이 충분히 크고 명확
4. **명확한 계층 구조**: 제목 > 설명 > 액션 버튼
5. **최소한의 텍스트**: 간결한 설명
6. **아이콘 중심**: 시각적 요소 강조
7. **전체 너비 버튼**: 하단 액션 버튼은 항상 전체 너비

## 현재 구현과의 차이점

| 요소 | 기존 디자인 | 현재 구현 |
|------|------------|----------|
| 배경색 | 순수 검은색 | 흰색/어두운 회색 |
| Primary 색상 | 보라색 (#8B7FF5) | 핑크색 (#FF6B9D) |
| 버튼 스타일 | 전체 너비, 둥근 사각형 | 원형 또는 다양한 크기 |
| 스타일 선택 | 그리드 카드 + 아이콘 | 텍스트 버튼만 |
| 결과 화면 | 좌우 슬라이더 | 토글 버튼 |
| 홈 버튼 | + 아이콘 버튼 | 카메라 이모지 원형 버튼 |

## 수정 필요 사항

1. **색상 테마 변경**: 핑크 → 보라색, 라이트 → 다크
2. **홈 화면**: 원형 버튼 → 둥근 사각형 + 아이콘
3. **스타일 선택**: 텍스트만 → 아이콘 + 텍스트 카드
4. **결과 화면**: 토글 → 좌우 슬라이더
5. **전체적인 레이아웃**: 더 넓은 여백, 큰 버튼
